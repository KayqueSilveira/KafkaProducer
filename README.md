# KafkaProducer


## Configuração do projeto
* **Java** 17
* **Spring boot** 2.7.1
* **Maven** 3.8+

## Ferramentas
* Docker
* Kafka
* Banco h2

## Sobre o projeto
* A finalidade do projeto é cadastrar um cliente no banco e assim que cadastrar o cliente, envia uma mensagem para o kafka
cujo a propriedade do kafka é RegistroEvento
